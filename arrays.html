Массивы в JavaScript

Массивы в JavaScript представляют собой объекты, которые используются для хранения упорядоченных коллекций данных. Они позволяют работать с множеством элементов, используя индексы.

Создание массива

Массив можно создать несколькими способами: через литерал массива или через конструктор Array.

// Литерал массива
let fruits = ["яблоко", "банан", "вишня"];

// Через конструктор
let numbers = new Array(10, 20, 30);
let emptyArray = new Array(5); // Массив с 5 undefined элементами

Доступ к элементам массива

Для доступа к элементам массива используется индекс, начиная с 0. Также можно изменять элементы массива напрямую.

let fruits = ["яблоко", "банан", "вишня"];
console.log(fruits[0]); // "яблоко"
fruits[1] = "апельсин"; // Замена элемента
console.log(fruits[1]); // "апельсин"

Метод push()

Метод push() добавляет один или несколько элементов в конец массива и возвращает новую длину массива.

let fruits = ["яблоко", "банан"];
fruits.push("вишня");
console.log(fruits); // ["яблоко", "банан", "вишня"]

Метод pop()

Метод pop() удаляет последний элемент массива и возвращает его.

let fruits = ["яблоко", "банан", "вишня"];
let lastFruit = fruits.pop();
console.log(lastFruit); // "вишня"
console.log(fruits); // ["яблоко", "банан"]

Метод shift()

Метод shift() удаляет первый элемент массива и возвращает его.

let fruits = ["яблоко", "банан", "вишня"];
let firstFruit = fruits.shift();
console.log(firstFruit); // "яблоко"
console.log(fruits); // ["банан", "вишня"]

Метод unshift()

Метод unshift() добавляет один или несколько элементов в начало массива.

let fruits = ["банан", "вишня"];
fruits.unshift("яблоко");
console.log(fruits); // ["яблоко", "банан", "вишня"]

Метод concat()

Метод concat() используется для объединения массивов или добавления значений к массиву.

let fruits = ["яблоко", "банан"];
let moreFruits = ["вишня", "апельсин"];
let allFruits = fruits.concat(moreFruits);
console.log(allFruits); // ["яблоко", "банан", "вишня", "апельсин"]

Метод slice()

Метод slice() возвращает новый массив, содержащий копию части исходного массива, без изменения оригинала.

let fruits = ["яблоко", "банан", "вишня", "апельсин"];
let slicedFruits = fruits.slice(1, 3);
console.log(slicedFruits); // ["банан", "вишня"]

Метод splice()

Метод splice() изменяет содержимое массива, удаляя или добавляя элементы.

let fruits = ["яблоко", "банан", "вишня"];
fruits.splice(1, 1, "апельсин");
console.log(fruits); // ["яблоко", "апельсин", "вишня"]

Метод forEach()

Метод forEach() выполняет переданную функцию для каждого элемента массива.

let fruits = ["яблоко", "банан", "вишня"];
fruits.forEach(function(fruit) {
    console.log(fruit);
});

Метод map()

Метод map() создает новый массив, в котором каждый элемент является результатом выполнения функции на соответствующем элементе исходного массива.

let numbers = [1, 2, 3];
let squaredNumbers = numbers.map(num => num ** 2);
console.log(squaredNumbers); // [1, 4, 9]

Метод filter()

Метод filter() создает новый массив, содержащий только те элементы, которые прошли проверку в переданной функции.

let numbers = [1, 2, 3, 4, 5];
let evenNumbers = numbers.filter(num => num % 2 === 0);
console.log(evenNumbers); // [2, 4]

Метод reduce()

Метод reduce() применяется для преобразования массива в одно значение (например, сумму).

let numbers = [1, 2, 3, 4];
let sum = numbers.reduce((accumulator, currentValue) => accumulator + currentValue, 0);
console.log(sum); // 10

Метод find()

Метод find() возвращает первый элемент массива, который удовлетворяет условию.

let numbers = [1, 2, 3, 4];
let found = numbers.find(num => num > 2);
console.log(found); // 3

Метод indexOf()

Метод indexOf() возвращает индекс первого элемента массива, который совпадает с заданным значением.

let fruits = ["яблоко", "банан", "вишня"];
let index = fruits.indexOf("банан");
console.log(index); // 1

Массивы и деструктуризация

Можно извлекать значения массива в переменные с помощью деструктуризации.

let fruits = ["яблоко", "банан", "вишня"];
let [first, second] = fruits;
console.log(first); // "яблоко"
console.log(second); // "банан"

Методы для работы с многомерными массивами

Многомерные массивы (например, матрицы) также могут быть обработаны с использованием обычных методов.

let matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
];

matrix.forEach(row => {
    console.log(row);
});
