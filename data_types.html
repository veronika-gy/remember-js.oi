Переменные и типы данных

Переменные используются для хранения данных. В JavaScript есть три способа объявления переменных:

// Объявление переменных
var oldWay = "Я устарел"; // Устаревший способ
let modern = "Я современный"; // Рекомендуется
const constant = "Я не изменяюсь"; // Константа

Типы данных

В JavaScript есть несколько типов данных:

  • Number — числовые значения
  • String — строки (текст)
  • Boolean — логические значения (true / false)
  • Object — сложные структуры данных
  • Array — массивы (списки значений)
  • Null и Undefined — отсутствие значения
// Примеры типов данных
let number = 42;        // Число
let text = "Привет";    // Строка
let isTrue = true;      // Логическое значение
let user = { name: "Иван", age: 30 }; // Объект
let numbers = [1, 2, 3]; // Массив

Числовой тип данных

//Пример
let num = 16;
num = 19.345;

Числовой тип данных(number) представляет как целочисленные значения, так и числа с плавающей точкой.

Существует множество операций для чисел, например, умножение *, деление /, сложение +, вычитание - и так далее.

Кроме обычных чисел, существуют так называемые «специальные числовые значения», которые относятся к этому типу данных: Infinity, -Infinity и NaN.

Infinity представляет собой математическую бесконечность ∞. Это особое значение, которое больше любого числа.

//Его можно получить с помощью деления:
alert( 1 / 0 ); // Infinity

//Или задать его явно:
alert( Infinity ); // Infinity

Значение NaN «прилипчиво». Любая математическая операция с NaN возвращает NaN:

//Примеры
alert( NaN + 1 ); // NaN
alert( 3 * NaN ); // NaN
alert( "не число" / 2 - 1 ); // NaN

Строковый тип данных

//Пример
let str1 = "Привет";
let str2 = 'Мир';
let str3 = `JavaScript`;

Строковый тип данных (string) используется для представления текстовой информации. Строки можно заключать в двойные кавычки " ", одинарные кавычки ' ', а также использовать обратные кавычки ` ` для шаблонных строк.

Строки поддерживают множество операций, таких как объединение с помощью +, получение длины через length, доступ к символам через индексы и встроенные методы.

//Конкатенация строк:
let greeting = "Привет, " + "мир!";
alert(greeting); // "Привет, мир!"

//Шаблонные строки:
let name = "Алекс";
alert(`Привет, ${name}!`); // "Привет, Алекс!"

Строки неизменяемы, что означает, что нельзя изменить символ в строке, но можно создать новую строку.

//Попытка изменить строку:
let text = "JavaScript";
text[0] = "j"; // Ошибка: строки неизменяемы

//Однако можно создать новую строку:
text = "javascript";

Логический тип данных

//Пример
let isTrue = true;
let isFalse = false;

Логический тип данных (boolean) имеет всего два значения: true (истина) и false (ложь). Чаще всего используется для условий и проверок.

//Пример использования в условии:
let isAdult = true;
if (isAdult) {
    alert("Доступ разрешён");
} else {
    alert("Доступ запрещён");
}

Булевы значения часто получаются в результате логических операций, таких как сравнение:

//Примеры сравнения:
alert( 10 > 5 );  // true
alert( 3 == 5 );  // false
alert( "яблоко" != "груша" ); // true

Также любое значение в JavaScript может быть приведено к логическому. Например:

//Приведение к логическому значению:
alert( Boolean(1) );    // true
alert( Boolean(0) );    // false
alert( Boolean("Привет") ); // true
alert( Boolean("") );   // false

Объектный тип данных

//Пример создания объекта
let user = {
    name: "Алекс",
    age: 25,
    isAdmin: true
};

Объект (object) — это коллекция свойств, где каждое свойство имеет ключ (имя) и значение. Объекты позволяют хранить структурированные данные.

//Доступ к свойствам объекта:
alert( user.name );  // "Алекс"
alert( user.age );   // 25

//Добавление нового свойства:
user.city = "Москва";
alert( user.city ); // "Москва"

В объекте могут храниться не только данные, но и функции (методы):

//Добавление метода в объект
user.sayHi = function() {
    alert("Привет!");
};
user.sayHi(); // "Привет!"

Для создания множества объектов можно использовать конструкторы или классы.

//Функция-конструктор
function User(name, age) {
    this.name = name;
    this.age = age;
}

let user1 = new User("Иван", 30);
let user2 = new User("Ольга", 28);

alert(user1.name); // "Иван"
alert(user2.age);  // 28

Массивы в JavaScript

//Пример создания массива
let fruits = ["Яблоко", "Банан", "Апельсин"];

Массив (array) — это упорядоченная коллекция элементов, к которым можно обращаться по индексу.

//Доступ к элементам массива
alert(fruits[0]); // "Яблоко"
alert(fruits[1]); // "Банан"

//Изменение элемента
fruits[2] = "Груша";
alert(fruits); // ["Яблоко", "Банан", "Груша"]

Массивы поддерживают множество методов, позволяющих добавлять, удалять и изменять элементы.

//Добавление и удаление элементов
fruits.push("Киви"); // Добавляет в конец
fruits.pop();        // Удаляет последний элемент

fruits.unshift("Ананас"); // Добавляет в начало
fruits.shift();           // Удаляет первый элемент

Перебор массива можно выполнить разными способами.

//Перебор с помощью for
for (let i = 0; i < fruits.length; i++) {
    alert(fruits[i]);
}

//Перебор с помощью for...of
for (let fruit of fruits) {
    alert(fruit);
}

//Метод forEach
fruits.forEach((item, index) => {
    alert(${index}: ${item});
});

Массивы также имеют полезные методы для работы с данными.

//Методы массивов
let numbers = [1, 2, 3, 4, 5];

//Создание нового массива с изменёнными элементами
let squared = numbers.map(num => num ** 2);
alert(squared); // [1, 4, 9, 16, 25]

//Фильтрация массива
let evenNumbers = numbers.filter(num => num % 2 === 0);
alert(evenNumbers); // [2, 4]

//Поиск элемента
let found = numbers.find(num => num > 3);
alert(found); // 4

Отсутствие значений: null и undefined

//Примеры
let emptyValue = null;
let notAssigned;

null и undefined представляют собой отсутствие значения, но имеют разные применения.

Null

null — это «ничего», «пусто» или «значение неизвестно». Обычно его присваивают явно.

//Пример использования null
let user = null;
alert(user); // null

Undefined

undefined означает, что значение не было присвоено.

//Пример использования undefined
let age;
alert(age); // undefined

Различия между null и undefined можно увидеть в операциях.

//Сравнение null и undefined
alert(null == undefined);  // true (нестрогое равенство)
alert(null === undefined); // false (строгое равенство)

//Арифметические операции
alert(null + 1); // 1 (null преобразуется в 0)
alert(undefined + 1); // NaN (undefined не может быть числом)

Использование ?? (оператор нулевого слияния) помогает работать с отсутствующими значениями.

//Оператор нулевого слияния ??
let value = null ?? "Значение по умолчанию";
alert(value); // "Значение по умолчанию"

let definedValue = 0 ?? "Значение по умолчанию";
alert(definedValue); // 0 (не null или undefined, поэтому не заменяется)
