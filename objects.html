Объекты в JavaScript

Объект (object) — это структура данных, состоящая из пар «ключ: значение».

Создание объекта

// Литерал объекта
  let user = {
      name: "Алекс",
      age: 25,
      isAdmin: true
  };
  
  // Через конструктор
  let user2 = new Object();
  user2.name = "Ольга";
  user2.age = 30;

Глубокое копирование

Чтобы создать независимую копию, используется structuredClone().

let original = { name: "Алекс", details: { age: 25 } };
  let clone = structuredClone(original);
  
  clone.details.age = 30;
  alert(original.details.age); // 25 (не изменилось)

Методы объекта

let person = {
      name: "Алекс",
      sayHi() {
          alert(Привет, меня зовут ${this.name});
      }
  };
  
  person.sayHi(); // "Привет, меня зовут Алекс"

Прототипное наследование

let animal = { eats: true };
  let rabbit = Object.create(animal);
  rabbit.jumps = true;
  
  alert(rabbit.eats); // true (унаследовано)

Классы

class User {
      constructor(name, age) {
          this.name = name;
          this.age = age;
      }
  
      sayHi() {
          alert(Привет, я ${this.name});
      }
  }
  
  let user = new User("Иван", 30);
  user.sayHi(); // "Привет, я Иван"

Расширение классов

class Animal {
      constructor(name) {
          this.name = name;
      }
  
      speak() {
          alert(${this.name} издаёт звук.);
      }
  }
  
  class Dog extends Animal {
      speak() {
          alert(${this.name} лает.);
      }
  }
  
  let dog = new Dog("Шарик");
  dog.speak(); // "Шарик лает."
